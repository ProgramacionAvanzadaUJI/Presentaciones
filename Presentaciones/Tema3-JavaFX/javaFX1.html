Qué es JavaFX

JavaFX es una biblioteca Java para crear Interfaces Gráficas de Usuario (Graphical User Interface - GUI).

Controles y Gestores de posicionamiento

Entre otras muchas cosas, JavaFX proporciona un conjunto de Controles, tales como botones, listas, tablas, etcétera, con los que crear GUIs.

Por su parte, los gestores de posicionamiento ubican los controles dentro de la ventana, liberándonos de la tarea de hacerlo componente a componente.
